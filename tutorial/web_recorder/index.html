Webレコーダー

これまで、Webレコーダー機能を用いると、Webページ操作を自動で記録することが可能になります。
ここでは、Webレコーダー機能の基本操作を学びます。

Webレコーダーの操作

  1. コンソールから新たにプロセスを作成します。

  2. Web Recorderを選択します。

  3. どのブラウザを使用するか選択します。
    ここではAutomation Browserを選択してください。

  4. Macro Recorderウィンドウが起動します

  5. URL欄にGoogleのURLを入力し、Goボタンを押下します。

    URL : http://google.co.jp/

  6. 検索キーワードに「RPA wikipedia」を設定し、エンターキーを押します。

  7. 検索結果ページが表示されます。

  8. もっとも上位に表示されたリンク上で右クリックし、「Extract Element Value > Text(“～～…“」を選択してください。

  9. 同様に2番目のページのリンク上で右クリックし、「Extract Element Value > Text(“～～…“」を選択してください。

  10. RECORDED ACTIONSから Show Preview を選択します。

  11. 2つの動作で対象ページの全タイトルが取得できたことを確認します。

  12. プレビュー画面を終了し、Finishボタンを押下します。

  13. 記録した内容がワークスペースに展開されます。

  14. 実行を行い、動作を確認します。

    Google Chrome、Mozilla Firefoxを利用する場合は、拡張機能のインストールが必要です。
