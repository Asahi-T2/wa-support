Task6 アプリケーションを操作してみましょう

問題

下記手順に従い、プロセスを作成してみましょう。（所要時間　20:00）

  1. 下記ファイルをダウンロードし、デスクトップに展開します。

    Task6.zip     Download

  2. 展開されたフォルダから「Task6.exe」を実行し、下記ユーザー情報を設定します。

ユーザーID : user1
パスワード : password

  1. [ログイン]ボタンを押下します。

  2. システムメニュー画面の[バックアップ]ボタンを押下します。

  3. バックアップが完了したメッセージを確認し、アプリケーションを終了します。

  4. 新たに「Task6」のプロセスを作成します。

  5. 上記操作内容をマクロレコーダーで記録します。

  6. 動作確認を行います。

    プログラムがうまく起動できない場合は、Run Applicationアクションを使用してプログラムを起動してください。
